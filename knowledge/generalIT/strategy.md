# 1. 经营策略

根据企业的理念来决定经营策略
1. 经营目标(Vision)选择
2. 商业环境分析     SWOT
3. 商业策略的定案   
4. 重要成功因素分析和抽出 CSF (Critical Sucess Factors)
5. 实行计划的策划 设置考核指标 KGI KPI


* Compliance 经营 : 遵守法律和社会伦理
* Core competence 经营 : 核心竞争力, 拥有其他公司所没有的特殊技术

竞争策略: 为了再特定的领取取得领先竞争公司的地位, 采取的策略
* cost leadership
* 差别化
* 集中战略

PPM 分析 (Products Portfolio Management)
